# パターンA：QRコード生成ツール（フロントのみ）

Next.js で動くシンプルなQRコード生成ツールです。

テキストやURLを入力すると、QRコードが生成されます。ブラウザだけで動作し、データベースは使いません。

## 対応する章

- 第1章：Claude CodeでQRコード生成ツールを作る
- 第3章：GitHub Pagesで公開する
- 第4章：Vercelで公開する
- 第5章：Cloudflare Pagesで公開する

## セットアップ

```bash
cd pattern-a-qr
npm install
npm run dev
```

ブラウザで http://localhost:3000 にアクセスして動作確認してください。
